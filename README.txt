Para rodar o projeto, incialmente precisa dos arquivos de dataset de treinamento e teste no mesmo diretorio do código da rede neural, salvo respectivamente como
"desafio_manutencao_preditiva_treino.csv" e "desafio_manutencao_preditiva_teste.csv".
Ter instalado na máquina, o Anaconda Navigator com a extensão Jupyter Notebook para conseguir abri o arquivo do código "CodigoRedeNeural.ipynb". 
É necessário ter instalado também o tensorflow para conseguir utilizar suas funções. Assim como outras extensões ditas no arquivo "requirements".
Feito isso, basta apenas rodar todos os códigos em sequencia para realizar novamente o tratamento dos dados,treinamento da rede neural, e predição final!
